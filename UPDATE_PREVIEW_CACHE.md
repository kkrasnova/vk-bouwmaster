# Как обновить превью ссылки в Telegram и других платформах

## Проблема
Telegram, WhatsApp, Facebook и другие платформы кэшируют превью ссылок. После изменения метатегов нужно очистить кэш.

## Решения

### 1. Использовать Telegram Bot для очистки кэша
1. Откройте бота: https://t.me/webpagebot
2. Отправьте ему ссылку на ваш сайт: `https://vkbouwmaster.com`
3. Бот обновит кэш превью

### 2. Использовать Facebook Debugger (работает и для Telegram)
1. Откройте: https://developers.facebook.com/tools/debug/
2. Вставьте URL: `https://vkbouwmaster.com`
3. Нажмите "Scrape Again" или "Обновить"
4. Это обновит кэш для Facebook, Telegram и WhatsApp

### 3. Добавить версию к URL (временное решение)
При отправке ссылки добавьте параметр версии:
- `https://vkbouwmaster.com?v=2`
- При следующем изменении используйте `?v=3` и т.д.

### 4. Пересобрать и перезапустить сайт
После изменения метатегов обязательно:
```bash
npm run build
npm run start
# или для разработки
npm run dev
```

## Важно
- Кэш может обновиться автоматически через 24-48 часов
- Для быстрого обновления используйте инструменты выше
- Убедитесь, что сайт доступен по адресу https://vkbouwmaster.com
